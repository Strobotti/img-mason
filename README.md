# ImgMason

_A light-weight masonry layout library_

## Install

[npm](https://www.npmjs.com/package/img-mason): `npm install img-mason --save`

## Initialize

With vanilla Javascript:

``` js
window.ImgMason('#masonry-element-with-imgs', { /* options */ });
```

HTML markup:
